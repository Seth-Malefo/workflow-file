## 📌 Project Overview:

This calculator supports:

- Real-time display updates

- Chain operations

# ✨ Features

# ✔ Number Input (0–9)
Users can input multi-digit numbers dynamically.

# ✔ Operators
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

# ✔ Equals (=)
Performs calculation and displays result.

# ✔ Clear (C)
Resets the calculator completely.

# ✔ Delete (DEL)
Removes the last digit entered.

# ✔ Decimal Support
Prevents multiple decimal points in a single number.

# ✔ Chain Calculations
Example:

5 + 5 + 5 =

Works seamlessly without resetting.

# ✔ Edge Case Handling
- Division by zero protection
- Prevents multiple operators
- Prevents multiple decimals
- Handles repeated equals press
