🚀 Week 3 – Responsive Landing Page.

📌 Overview

This project showcases a fully responsive landing page built using modern CSS layout techniques. The primary objective was to implement a mobile-first design strategy while combining CSS Grid and Flexbox to create a scalable and maintainable layout system.

The project demonstrates structured thinking, clean code practices, and responsive UI implementation.

🛠 Technologies Used

HTML5 (Semantic Structure)

CSS3

CSS Grid

Flexbox

Media Queries

CSS Transitions

🏗 Layout Architecture
🔹 CSS Grid – Structural Layout

CSS Grid controls the overall page structure and section alignment. It manages:

Feature section layout

Testimonial alignment

Responsive column distribution

Grid ensures predictable and scalable layout behavior across breakpoints.

🔹 Flexbox – Component Layout

Flexbox is used for internal component alignment, including:

Navigation bar (.navbar, .nav-links)

Card alignment within sections

Responsive directional changes

Flexbox allows flexible distribution and alignment of UI elements.

📱 Responsive Strategy (Mobile-First)

The project follows a mobile-first approach:

Base styles are optimized for small screens.

Layout enhancements are progressively applied using min-width media queries.

Breakpoints Implemented

600px → Large mobile devices

768px → Tablet

1024px → Desktop

This ensures smooth scaling across devices while maintaining performance efficiency.
